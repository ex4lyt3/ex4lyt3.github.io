TryHackMe's Pre-Security Path Review

17/7/21

Posted by ex4lyt3

Wanted to start on your cybersecurity journey but you don't know how? Well, TryHackMe's Pre-Security Path offers aspiring newbies a place to start off their journey with the knowledge before learning cybersecurity to enable them to start off their journey with the right foot.

What is TryHackMe?

It is an online learning platform for cybersecurity through the usage of rooms which is teaching content aimed at a specific topic in cybersecurity

Path Overview

The Pre-Security Path consists of several sections, such as Network Fundamentals, How The Web Works, Linux and Windows Fundamentals. All these sections are your fundamentals and basics you need before you get to hacking!

Thoughts & Feelings about Pre-Security

The Pre-Security Path covers a wide array of topics which all require you to understand these topics before proceeding on to hacking. It is a wonderful learning tool as you are able to learn security fundamentals in just one neatly arranged "path" and get started on security itself when you're done!

Conclusion

In cybersecurity, there are a ton of fundamentals you need before you can get started on cybersecurity itself.
Want to use nmap(recon tool)? Well, you first need to know how to use the terminal, a basic understanding of networking and how the web works.
Thus, it is essential to learn the basics before proceeding to the next level. Go on, and start your journey with TryHackMe!
